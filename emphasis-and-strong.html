Emphasis (<em>) and Strong (<strong>)

Semantic Emphasis

When you want to stress a word in a sentence, use emphasis to show that the word should be read with more weight or a different tone.

When something is of great importance, use <strong> to indicate that meaning.

Visual Only (CSS)

You can make text look italic or look bold using CSS, but this does not convey any extra meaning to assistive technologies.

The best practice is to use semantic elements first and then style them with CSS, instead of using generic elements purely for style.
